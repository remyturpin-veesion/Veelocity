# v0.6 Developer Statistics

> **For Claude:** REQUIRED SUB-SKILL: Use superpowers:executing-plans to implement this plan task-by-task.

**Goal:** Add per-developer statistics and leaderboard to track individual contributions.

---

## Task 1: Create DeveloperStatsService

**Files:**
- Create: `backend/app/services/metrics/developers.py`

**Metrics per developer:**
- PRs created (total, merged, open)
- Reviews given
- Comments made
- Avg time to merge (for their PRs)
- Lines changed (additions + deletions)

---

## Task 2: Create Developer API Endpoints

**Files:**
- Create: `backend/app/api/v1/endpoints/developers.py`
- Modify: `backend/app/api/v1/router.py`

**Endpoints:**
- `GET /developers` - List all developers with basic stats
- `GET /developers/{login}/stats` - Detailed stats for one developer
- `GET /developers/leaderboard` - Ranked by metric
